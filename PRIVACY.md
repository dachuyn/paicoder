# Privacy Policy for pAiCoder

**Effective Date:** 06/25/2026

## Overview

pAiCoder is a local-first developer tool available as a VSCode extension and CLI. It operates without a centralized backend and does not require user accounts.

We are committed to protecting your privacy. This policy explains what data is and is not collected.

---

## 1. Data We Do Not Collect

pAiCoder does not collect, store, or transmit:

* Source code from your projects
* Prompts or instructions you enter
* API keys (OpenAI, Anthropic, xAI, etc.)
* File contents from your workspace
* Personal information
* Usage logs tied to your identity

There is no user tracking, analytics backend, or telemetry system.

---

## 2. Local Data Storage

All configuration and sensitive data remain on your device, including:

* API keys (stored locally via VSCode settings or system keychain)
* Extension preferences
* Local CLI configuration

You are responsible for securing your own environment.

---

## 3. Third-Party AI Providers

pAiCoder sends user-provided inputs directly to third-party AI providers selected by the user, such as:

* OpenAI
* Anthropic
* xAI

These requests are made using your own API keys. Data sent to these providers is governed by their respective privacy policies.

pAiCoder does not proxy, log, or store these requests.

---

## 4. No Cloud Processing

pAiCoder operates entirely on your local machine.

We do not:

* Host user code
* Run inference servers
* Store conversations or project data
* Act as a data intermediary

---

## 5. Security Disclaimer

While pAiCoder does not collect data, you are responsible for:

* Safely storing your API keys
* Avoiding inclusion of sensitive secrets in prompts
* Securing your local development environment

---

## 6. Changes to This Policy

This Privacy Policy may be updated as new features are introduced (e.g., optional cloud services or collaboration features). Any updates will be posted in the repository or official website.

---

For questions, contact: supports@paicoder.com
